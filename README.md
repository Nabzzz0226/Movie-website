# MovieFlix 🎬

MovieFlix is a simple movie watching web application built using HTML, CSS, and JavaScript.

It displays popular movies with a modern UI and includes a search feature to filter movies.

---

# Features

- Modern movie website UI
- Responsive design
- Search movies functionality
- Movie cards with images
- Watch movie button
- Pure HTML, CSS, and JavaScript
